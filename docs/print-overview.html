1. Executive summary

Memory Words is a full-stack memory game: pick one of 25 categories, memorize words under time pressure, then find them in a grid with topic-matched distractors.

6. Admin authentication

Configure in backend/.env:

  • ADMIN_EMAIL — login email
  • ADMIN_PASSWORD — bcrypt-hashed on save
  • JWT_SECRET — token signing key
  • FRONTEND_URL — reset link base URL
  • SMTP_* — optional email for forgot password

Flow: Login → JWT in sessionStorage → Bearer token → protected inbox routes.
